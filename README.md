# AI Study Planner

A simple command-line study planner built in Python.  
This project helps you organize study tasks, track progress, and mark completion — all stored locally in a JSON file.

---

##  Features
- Add study tasks with subject and hours.
- View all tasks with progress indicators.
- Mark tasks as completed.
- Data persistence using `tasks.json`.

##  Installation
Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/study_planner_ai.git
cd study_planner_ai
pip install -r requirements.txt
```

*(Note: This project uses only Python’s standard library, so `requirements.txt` is minimal.)*

---

## Usage
Run the planner from the terminal:

```bash
python study_planner_ai.py
```

Available commands inside the app:
- `add` → Add a new study task
- `list` → Show all tasks
- `done` → Mark a task as complete
- `quit` → Exit the planner
